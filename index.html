Luminar RTS LiDAR Prototype

This project demonstrates a real-time embedded system using an ESP32 and FreeRTOS. It simulates a LiDAR-based hazard detection system for autonomous vehicles. The system monitors sensor input, detects hazards, and triggers emergency braking while maintaining real-time guarantees through task prioritization and synchronization.

🎥 Demo Video

🔧 Features

  • FreeRTOS multi-tasking system
  • Hard real-time hazard detection
  • ISR-triggered emergency braking
  • Queue, semaphores, and mutex synchronization
  • LED-based system feedback
